# 金融风控系统权威产品梳理

> 整理日期：2026-07-18
> 用途：辅助"多模态大模型在风控领域的实现"研究，定位成熟可用的权威产品/平台
> "权威"判定依据：第三方机构认证（Gartner / Forrester / IDC / 信通院）、行业榜单排名、大规模标杆客户落地

---

## 一、互联网大厂系（自带海量业务锤炼 → 技术外溢）

这类厂商的核心优势是**自有的支付/信贷/电商业务本身就是最高强度的风控练兵场**，再把沉淀的能力以云服务/平台形式输出。

| 厂商 | 代表产品 | 能力亮点 | 权威背书 / 落地 |
|---|---|---|---|
| **蚂蚁集团** | 蚁盾风控、支付宝/网商银行风控体系、图计算反洗钱 | 亿级用户实时交易风控；图计算构建上百亿节点关系网络；AI + 图计算双视角反洗钱 | 覆盖信贷/支付/反洗钱全链路 |
| **腾讯云天御** ⭐ | T-Sec 天御风控平台(RCP)、朴智平台(MaaS)、金融风控大模型、信鸽 | 一站式端到端风控（数据→特征→规则模型→场景）；朴智平台为行业首个持牌 MaaS，建模周期 1-2 月→T+2；金融风控大模型小样本定制，KS 提升 20%；信鸽解决合规数据流通 | **Gartner** 连续入选（2020-2024 在线/银行业欺诈检测指南）；**Forrester Wave 亚太反欺诈"领导者"**；覆盖金融 80% 标杆客户、7000+ 企业、拦截超 5000 万次可疑交易 |
| **京东科技** | 数智化风控大模型、供应链多模态单据识别 | 票据/合同/仓单等多模态单据结构化识别与交叉验证 | 依托京东供应链生态 |
| **百度智能云** | 商业银行智能风控平台解决方案 | "云智一体"，服务工行、浦发等 | 服务多家头部银行 |
| **奇富科技** | 信贷超级智能体、Qfin-VL-Instruct 多模态大模型 | 信贷全链路智能体化；多模态证件/流水/合同理解（FCMBench 综合第一） | 信贷垂直场景领先 |
| **阿里云** | 金融风控解决方案 | 依托通义千问大模型能力输出 | 集团生态协同 |

**与多模态方向最相关**：京东多模态单据、奇富信贷超级智能体、腾讯天御金融风控大模型、蚂蚁图计算+多模态。

## 二、专业第三方风控厂商（决策智能 / 反欺诈中台）

这类是**独立风控技术供应商**，产品化程度高，适合银行/消金/保险直接采购或联合运营。

| 厂商 | 代表产品 | 能力亮点 | 权威背书 / 落地 |
|---|---|---|---|
| **同盾科技** ⭐ | 天策-决策引擎、云图-知识图谱、明模-模型平台、**诸葛-金融风控大模型** | 决策智能产品矩阵；知识图谱挖团伙欺诈；大模型集成智能决策引擎/知识构建/特征挖掘/**多模态服务**五大能力；"记忆-遗忘"机制免全量重训 | 服务数百家金融机构；与数十家大型银行合作；某股份行预警涉案 2.6 亿元大案（准确率 90%）；某行拦截涉诈资金 6.7 亿 |
| **邦盛科技** | 实时风控决策引擎 | 毫秒级实时交易反欺诈 | 金融行业广泛落地 |
| **顶象技术** | 业务安全风控、全链路反诈 | 设备指纹、无感验证、风控引擎 | 银行/互金覆盖 |
| **百融云创** | 智能分析与决策平台 | 信用评分、反欺诈、精准营销 | 上市金融科技公司 |
| **冰鉴科技** | 智能风控决策中台 | 中小微企业/个人信用评估 | 多家金融机构 |
| **数美科技** | 天网-智能业务风控、天净-智能内容风控 | 全栈式实时业务风控 + 内容风控 | 在线业务风控头部 |
| **中关村科金·得助智能** | 实时智慧风控平台 | 贷前/贷中/贷后全流程；决策中心（决策流/树/表/集）、进件中心、流程中心、数据中心、验证中心、运维中心、监控中心七大模块；平均 <100ms | 提供 14 天免费试用，配置化自助作业 |
| **睿智科技 / 融慧金科 / 萨摩耶数科 / 马上消费** | 一站式风控赋能 / 定制风控 / 金融云 / 消费金融智能风控 | 细分场景深耕 | 榜单常客 |

**与多模态方向最相关**：同盾"诸葛金融风控大模型"明确集成多模态服务能力；中关村科金提供完整可视化决策平台可作为自研底座参考。

| 机构 | 代表能力 | 特点 |
|---|---|---|
| **微众银行** | 联邦学习风控中台、相机指纹防欺诈 | 数字银行原生，隐私计算领先 |

| 厂商 | 产品 | 价值 |
|---|---|---|
| **魔数智擎** | 金融模型风险管理解决方案 | "战略-业务-产品"三级模型风险管理体系，大小模型融合统一管理底座；应对监管对模型风险（"第九大风险"）要求；已在近 60 家大型金融机构落地 |

## 六、给你（多模态风控研究方向）的选型建议

1. **想看"大模型 + 多模态"落地范式** → 优先研究：奇富 Qfin-VL-Instruct / 信贷超级智能体、京东多模态单据、新网多模态智控平台、腾讯天御金融风控大模型、同盾诸葛大模型（多模态服务）。
2. **想看"工程化平台 + 决策引擎"底座** → 中关村科金得助智能（七大中心结构清晰，可照着画自己的系统架构）、同盾天策决策引擎 + 云图知识图谱。
3. **想看"权威认证 + 大规模金融客户"** → 腾讯云天御（Gartner/Forrester 双认证）、蚂蚁蚁盾（亿级用户实战）。
4. **想看"模型合规治理"** → 魔数智擎模型风险管理。

> 注：本表基于公开搜索结果整理（2025-2026 年公开报道、厂商官网、行业榜单），产品能力与定位会随版本迭代变化，落地前建议以厂商最新官方文档为准。
