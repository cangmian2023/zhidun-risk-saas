## 一、零售信贷风控

### 1.1 贷前审核

| 模块 | 页面 | 说明 |
|------|------|------|
| **申贷审核** | 申贷审核 | 保持不变 |
| | 申贷审核报告详情页 | 不需要添加左侧菜单，页面报错不变 |
| **信息核验** | 信息核验 | 保持不变 |
| | 信息核验报告详情页 | 不需要添加左侧菜单，页面报错不变 |
| | 核验规则 | 配置信息核验的校验规则、阈值、开关 |
| **信用风控** | 信用风控 | 保持不变 |
| | 信用风控报告详情页 | 不需要添加左侧菜单，页面报错不 |
| | 信用模型 | 配置六大维度的权重、评分规则、叠加惩罚机制 |
| **欺诈识别** | 欺诈识别 |  保持不变 |
| | 欺诈识别报告详情页 |  不需要添加左侧菜单，页面报错不变 |
| | 反欺诈规则库 | 管理18条反欺诈规则，配置规则权重、命中条件、处置建议 |
| | 黑名单管理 | 管理手机号、设备指纹、身份证号、银行卡黑名单 |
| | 团伙库管理 | 管理已知欺诈团伙信息、团伙成员、团伙特征 |
| **决策报告** | 决策报告 | 展示所有生成的决策报告，支持按报告类型、决策建议筛选 |
| | 决策报告详情页 | 整合信息核验+信用风控+欺诈识别的综合报告，输出最终决策建议 |
| | 报告模板 | 配置决策报告的展示模板、字段、样式 |

### 1.2 贷中监控

| 模块 | 页面 | 说明 |
|------|------|------|
| **监控任务管理** | 监控任务 | 展示所有监控任务，支持按产品、场景、频次筛选 |
| | 监控任务 | 创建/编辑监控任务，配置扫描频次、客群范围、预警规则 |
| | 监控任务执行日志 | 展示监控任务的执行历史、耗时、扫描客群数 |
| **红黄灯预警** | 预警 | 展示所有红黄灯预警记录，支持按预警等级、监控场景筛选 |
| | 预警详情页 | 展示预警触发规则、风险评分变化、建议处置方式 |
| | 预警规则 | 配置红灯/黄灯预警的触发条件、阈值、通知方式 |
| **客群风险看板** | 客群风险 | 展示各客群的实时风险分布、趋势变化 |
| | 单客风险 | 追踪单个客户的风险评分变化历史、预警记录 |
| | 风险趋势 | 展示客群风险评分的趋势变化、同比环比分析 |
| **处置管理** | 处置任务 | 展示待处置的预警任务，支持分配、转派、批量处置 |
| | 处置 | 展示所有处置历史，含处置人、处置时间、处置结果 |
| | 处置策略 | 配置自动处置策略（如自动降额、自动冻结） |
| **结果输出** | API接口 | 管理API接口的调用权限、限流、计费 |
| | 推送 | 配置URL推送、文件交换的接收地址、格式、频次 |
| | 下载中心 | 支持按时间范围、客群范围下载监控结果文件 |
